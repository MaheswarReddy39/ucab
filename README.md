# 🚖 UCab Booking System

UCab Booking System is a Full Stack MERN application developed as a group project. The project allows users to register, book cabs, make payments, receive ride notifications, and manage bookings. It also provides an Admin Panel to manage users, drivers, cabs, and bookings.

---

# 📖 Project Overview

The main objective of this project is to understand how an online cab booking system works by implementing both frontend and backend using the MERN Stack.

The application includes:

- User Authentication
- Driver Management
- Cab Management
- Ride Booking
- Automatic Driver Assignment
- Fare Calculation
- Payment Module
- Notification System
- Admin Dashboard
